# Accounting Review Skill #

To try this one, point your Claude Cowork folder to the "review agent" folder from your resources.

The review agent contains a skill, and demo data from Xero. 

Once you've selected the folder in Cowork type the prompt

``` Find anomalies in these reports ```

The heavy lifting of the prompt is actually handled by the skill, so even though are prompt is amiguous the skill will get triggered.

You will soon receive a result based on what the skill was able to find. 

### Adjusting the Skill 

The skill currently only uses these reports

- Balance Sheet
- Profit and Loss
- Aged Receivables Summary
- Aged Payables Summary
- Account Transactions

If there are any you would like to add, you can now use this skill as a starting template
